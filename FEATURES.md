## 📦 快速开始

### 🔧 安装步骤

1. **下载扩展** - 从 Chrome Web Store 安装
2. **设置快捷键** - 访问 chrome://extensions/shortcuts 配置
3. **开始使用** - 按快捷键或点击工具栏图标

### 🎯 使用技巧

- **快捷键唤起** - Ctrl+Shift+S (Mac: Cmd+Shift+S)
- **Tab 键补全** - 输入时按 Tab 键自动补全
- **域名过滤** - 点击域名标签快速筛选
